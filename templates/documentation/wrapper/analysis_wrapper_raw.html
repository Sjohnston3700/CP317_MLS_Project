Entity Objects

  • Host- Stores host information from config file.
    • Name
    • Url
    • Scheme
      • <\ul>
    • User - Stores user authentication credentials.
      • Name
      • Laurier ID #
      • Login Credentials
    • Course - Stores information about a course.
      • Name
      • Access
      • Active
      • ID
      • Grade Items
    • Student - Stores information about a student.
      • Name
      • University ID
      • ID
    • Grade Item - Stores information about a grade item. Abstract Class. Implimented by specific grade item type
      • Name
      • ID
      • Type
    • Numeric Grade Item
      • Out of
    • Grade - Stores information about a specific grade. Abstract Class. Implimented by specific grade type
      • Student
      • Grade Item
    • Numeric Grade - Stores information about a numeric grade
      • Value
      • Public feedback
      • Private feedback
    User has 0 to many courses. Course has 0 to many grade items and 0 to many students. Grade has 1 student and 1 grade item.
