1. Introduction

The Brightspace API Wrapper is a Python library used to make calls to the Brightspace API. In essence, it allows developers to fetch and push information through the API in a much more friendly manner. Rather than directly making HTTP requests, developers can simply use the functions available in the library. The wrapper is being developed primarily for ezMarker, a web application for uploading grades and feedback to a Brightspace environment.

1.1 Purpose

The purpose of this document is to outline the requirements for building the Brightspace API Wrapper. It lists the API requests the wrapper will contain.

1.2 Scope

[explain what the wrapper is and what is does and why it is good in slightly more info than introduction]

The wrapper will only include the requests needed by ezMarker. The scope may later be increased, time-permitting, to include other key API requests, for the purpose of making the wrapper a useful resource for future development projects.

1.3 Definitions

Brightspace: A learning management system produced by Desire2Learn. It allows students and instructors to communicate, share resources, and keep track of student progress. A vital part of this system is the feature that allows instructors to set grades and give feedback on student work. The Brightspace API provides functionality for interacting with Brightspace environments.

Desire to Learn (D2L): A Waterloo-based company that produces learning management software. They are the creators of Brightspace.

3. Software Requirements

This section lists the API calls that the wrapper contains.

3.1 User Login

Two function calls to retrieve user credential:

	d2lauth.fashion_app_context(app_id=app_creds['app_id'], app_key=app_creds['app_key'])

	create_url_for_authentication(app_config["lms_host"], app_url)

3.2 User Logout

The following URL handles logout:

	https://{host}/d2l/logout

3.3 Check API Version

GET /d2l/api/(productCode)/versions/(version)

	Determine if a specific product component supports a particular API version.

3.4 Get Accessible List of Courses

GET /d2l/api/lp/(version)/enrollments/myenrollments/

	Retrieve the list of all enrollments for the current user.

GET /d2l/api/lp/(version)/enrollments/users/(userId)/orgUnits/

	Retrieve a list of all enrollments for the provided user.

GET /d2l/api/lp/(version)/roles/(roleId)

	Retrieve a particular user role.

3.5 Get Class List

GET /d2l/api/lp/(version)/enrollments/orgUnits/(orgUnitId)/users/

	Retrieve the collection of users enrolled in the identified org unit.

GET /d2l/api/le/(version)/(orgUnitId)/classlist/

	Retrieve the enrolled users in the classlist for an org unit.

3.6 Get Grade Items

GET /d2l/api/le/(version)/(orgUnitId)/grades/

	Retrieve all the current grade objects for a particular org unit.

3.7 Set Specific Grade Item

PUT /d2l/api/le/(version)/(orgUnitId)/grades/(gradeObjectId)

	Update a specific grade object.
