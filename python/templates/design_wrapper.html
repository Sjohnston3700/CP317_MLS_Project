Introduction

The Brightspace API Wrapper is a library used to make calls to the Brightspace API, which supports the Python and PHP versions of ezMarker. The functions in the wrapper's library will be used rather than directly making HTTP requests. This will allow developers to fetch and push information through the API in a friendly manner. The wrapper is being developed primarily for ezMarker, which allows for uploading grades and feedback to a Brightspace environment.
